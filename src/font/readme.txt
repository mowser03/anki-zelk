STEPS TO ENABLE CUSTOM FONT 
For windows :- 
1. Hit Windows + R 
2. Enter following in run prompt : %APPDATA%\Anki2
3. Go to user1 / (your profile name)
4. paste  _HelveticaNeue.otf there 
More help at : https://docs.ankiweb.net/templates/styling.html#installing-fonts
